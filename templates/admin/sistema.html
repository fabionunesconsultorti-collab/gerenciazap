Backup de Dados

Gere um arquivo compacto contendo o banco de dados completo (SQLite) e todos os arquivos de mídia (uploads) para salvaguarda externa.

Restauração de Sistema

Recupere o sistema utilizando um arquivo .zip gerado anteriormente. Aviso: Isto substituirá o banco de dados atual.

Verificação de Consistência

Realiza uma checagem profunda de integridade estrutural (PRAGMA integrity_check) no banco de dados para evitar corrupção de tabelas e índices.

Atualização Plataforma

Sincronize a plataforma local com o repositório oficial no GitHub (`gerenciazap`). Recomendável fazer backup antes da atualização.
